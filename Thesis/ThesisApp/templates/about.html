เกี่ยวกับ

     ระบบนี้เป็นส่วนหนึ่งของเว็บแอปพลิเคชันที่มีการใช้งานภาษา Python และโครงสร้าง HTML ของ Django ในการพัฒนา เว็บนี้มีวัตถุประสงค์เพื่อแสดงข้อมูลของบุคคลที่ใช้งานเว็บ ประกอบด้วยบุคคลผู้ปกครองและเด็ก และมีการใช้งาน Web Speech API เพื่อให้อ่านข้อความเสียงออกมา.

การทำงานของระบบ:

  1. หน้าเว็บ "หน้าแรก" นี้เป็นส่วนย่อยของ template "base.html" ซึ่งมีโครงสร้างหน้าเว็บที่เหมือนกันในหลายหน้า เพื่อให้เว็บไซต์มีลักษณะเดียวกันทั้งหมด.
  2. เมื่อหน้าเว็บโหลดเสร็จสมบูรณ์ (load) ระบบจะเริ่มทำงานในฟังก์ชัน playSpeechAutomatically() ที่ถูกเรียกขึ้นอัตโนมัติ การทำงานของฟังก์ชันนี้คืออ่านข้อความเสียงโดยใช้ Web Speech API และส่งคำร้องขอ AJAX เพื่ออัปเดตสถานะ "Called" ของบุคคลที่เกี่ยวข้อง.
  3. ฟังก์ชัน speak(text, personId, repeatCount) ใช้สำหรับอ่านข้อความเสียงโดยกำหนดข้อความที่ต้องการอ่าน ไอดีของบุคคล และจำนวนครั้งในการอ่าน เมื่อเรียกใช้ฟังก์ชันนี้ Web Speech API จะทำการอ่านข้อความที่กำหนดด้วยเสียงในภาษาไทย (th-TH).
  4. หน้าเว็บจะแสดงข้อมูลของบุคคลที่เกี่ยวข้องโดยแสดงภาพและชื่อของผู้ปกครองและเด็ก ในส่วนของภาพและชื่อของผู้ปกครองและเด็ก มาจากข้อมูลที่ถูกส่งมาจากตัวแปร all_person ที่อยู่ในตัวแปร Datatext.
  5. หน้าเว็บยังมีฟอร์มสำหรับกรอกหมายเลขผู้ปกครองของท่าน ซึ่งหากกรอกหมายเลขและกด "Search" จะทำการส่งคำร้องขอ (POST request) ไปยัง server เพื่อค้นหาข้อมูลของผู้ปกครองที่เกี่ยวข้องและแสดงข้อมูลของผู้ปกครองนั้นในหน้าเว็บ.
  6. รวมถึงระบบนี้ยังมีการใช้งาน CSRF token เพื่อรักษาความปลอดภัยในการทำร้องขอ AJAX และมีการใช้งาน CSS เพื่อกำหนดความกว้างของส่วนของฟอร์มและการจัดวางเนื้อหาในหน้าเว็บให้เป็นระเบียบเรียบร้อย.
